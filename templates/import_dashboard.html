Guía de Importación

  • Excel (.xlsx): Formato moderno de Microsoft Excel
  • Excel 97-2003 (.xls): Formato legacy
  • CSV (.csv): Valores separados por comas

Para Equipos Individuales:
  • Código Activo (obligatorio)
  • Serial
  • Marca
  • Modelo
  • Estado
  • Usuario Asignado
  • Observaciones
Para Usuarios:
  • Cédula (obligatorio)
  • Nombre (obligatorio)
  • Email (obligatorio)
  • Cargo
  • Departamento

El sistema reconoce automáticamente variaciones en los nombres de columnas:

  • "Código Activo" → codigo_barras_individual
  • "Número Serial" → serial
  • "Sistema Operativo" → so
  • "Usuario Asignado" → asignado_nuevo

  • Duplicados: Se omiten automáticamente
  • Columnas faltantes: Solo se importan las que existen en BD
  • Errores por fila: Se reportan pero no detienen la importación
  • Tipos de datos: Se convierten automáticamente
